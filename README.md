# React Minimal Boilerplate

A very lean React boilerplate, using webpack.

## Production

```
npm install
npm run webpack
```

## Development

```
npm install
npm run dev
```

## File structure
```
public/
    js/
        bundle.js
    index.html
src/
    components/
        Item.js
        List.js
        Root.js
    styles/
        styles.scss
    index.js
package.json
webpack.config.js
```

## Included libraries
- [Webpack](http://webpack.github.io/)
- [Babel](https://babeljs.io/)
- [React](http://facebook.github.io/react/)
